## 📌 Featured Projects

### 🏗️ [AWS Lakehouse Pipeline](https://github.com/vyshnavi-nandyala/aws-lakehouse-pipeline)
Serverless, production-grade data lake on AWS S3 using Glue, Lambda, and Athena.  
**Stack:** PySpark · AWS Glue · S3 · Airflow DAGs · dbt models · GitHub Actions CI  
Ingests raw data, applies medallion architecture (Bronze → Silver → Gold), and exposes curated datasets via Athena for BI consumption.

---

### 🤖 RAG Pipeline — AI-Powered Document Q&A 
End-to-end Retrieval-Augmented Generation pipeline for intelligent document querying.  
**Stack:** LlamaIndex · pgvector · Claude Sonnet (Anthropic API) · OpenAI Embeddings · FastAPI  
Chunks and embeds documents into a vector store, retrieves context semantically, and generates precise LLM-grounded answers.

---

### 🔍 JobForge — AI Job Search Agent
AI agent that scrapes major job boards for Data Engineer roles, auto-ranks by match score, and tailors resumes per JD in `.docx` format.  
**Stack:** Python · Claude API · BeautifulSoup · FastAPI · React Dashboard · Email Notifications  
Features a semi-automated review workflow so the user retains final control over every application.
